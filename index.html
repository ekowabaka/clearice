What is ClearICE?

ClearIce is a library intended to help you supercharge your PHP CLI app development. ClearIce helps PHP CLI applications to parse command line arguments which are presented in a style similar to what you would find in most GNU applications. It also allows you to perform simple I/O operations such as outputing text to standard output (with the capability of controlling output verbosity levels) and reading input from the standard input (with the capability of interactively validating input). Another cool feature of ClearIce is that it can automatically generate help texts for your apps.

Check out the documentation to find out more

Features

A summary of what ClearIce can currently do.

  • Command line argument parsing.
  • Command line argument validation.
  • Support for grouping valid arguments under specific commands.
  • Automatic generation of help messages
  • Automatic instantiation of Command classes for CLI applications (Framework?)
  • Interactive console I/O with input validation.
